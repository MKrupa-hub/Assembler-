Laboratoria 29.03.2021

1. Program służy do mnożenia dwóch liczb hexadecymalnych.

2. Aby skompilować program należy umieścić plis lab1.s z makefile w tym samym folderze a następnie w konsoli napisać "make"(można na koniec napisać "make clean" aby usunąć stworzone do kompilacji pliki).

3. Po wpisaniu ./lab1 program sie uruchomi i będzie czekał na podania zdania które ma zaszyfrować, po zatwierdzeniu klaiwszem "enter" program wypisze już zaszyfrowaną wiadomość.
